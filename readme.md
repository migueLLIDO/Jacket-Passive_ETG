# Jacket - Aesthetic of Vice

A passive item for Jacket character in Enter the Gungeon, inspired by Hotline Miami's aggressive gameplay.

## How it works

Kill enemies quickly to gain stacks (max 25):

- **+1% Damage** per stack (max 25%)
- **+0.5% Movement Speed** per stack (max 12.5%)
- **+1% Reload Speed** per stack (max 25%)

**Taking damage resets ALL stacks!**

Stacks last 5 seconds. Kill another enemy to refresh.

## Dependencies

- BepInExPack
- Jacket CC
